Important sequencing note
-------------------------
Run the top-level Nextflow analysis commands one at a time in the same clone.
A bare -resume uses the clone's shared .nextflow session history. Launching two
-resume commands concurrently can make the second command select the active
session and report a lock. This is a test-scheduling collision, not an
OncoTracer analysis failure.
