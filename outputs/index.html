    Outputs¶

    All workflow outputs are written under outdir.

    <outdir>/
  01_samurai_ont/
  02_bam_refinement/
  03_cna_codification/
  04_cna_custom_plots/
  05_cna_classifier/
  06_workflow_summary/

    01_samurai_ont¶

    Present only when starting ONT from FASTQ/barcodes.

    Contains SAMURAI/ONT caller outputs generated before refinement.

    If you start from existing ONT ont_ichor_dir, this folder is skipped.

    02_bam_refinement¶

    Contains BAM-supported boundary refinement outputs.

    Dataset-specific folders look like:

    02_bam_refinement/ONT_ichorcna_500kb/
02_bam_refinement/illumina_qdnaseq_100kb/

    Important files:

    04_final_results/final_segments.tsv
04_final_results/cna_cytogenomic_input/qdnaseq_bins/
01_tables/refined_bins.tsv.gz

    final_segments.tsv is the refined segment table. qdnaseq_bins/ is the converter-ready bin directory used by the codification stage. refined_bins.tsv.gz is used by custom plotting for log2 profile figures.

    03_cna_codification¶

    Contains standardized CNA event tables.

    03_cna_codification/cna_events.tsv
03_cna_codification/cna_cytogenomic_notation.tsv

    cna_events.tsv is the main machine-readable CNA event table. cna_cytogenomic_notation.tsv contains cytogenomic-style descriptions.

    04_cna_custom_plots¶

    Contains plots generated by cna_codification/scripts/plot_cna_events.py.

    Common files:

    04_cna_custom_plots/cna_genome_overview.pdf
04_cna_custom_plots/cna_log2_ratio_profiles_all_samples.pdf
04_cna_custom_plots/cna_per_sample_pages.pdf
04_cna_custom_plots/cna_event_burden_by_sample.pdf
04_cna_custom_plots/cna_event_counts_by_sample.pdf
04_cna_custom_plots/cna_recurrent_cytobands.pdf
04_cna_custom_plots/cna_frequency_gene_panels.pdf
04_cna_custom_plots/per_sample_log2_cna/*.pdf

    These are intended for visual inspection of event burden, recurrent bands, genome-wide profiles, and per-sample log2 CNA profiles.

    05_cna_classifier¶

    Present only when run_cna_classifier: true.

    Important subfolders:

    01_prepared/
02_classification/
03_report/
04_gistic/
06_knowledge/
07_pathology/

    Common files:

    05_cna_classifier/02_classification/cna_patient_classification.tsv
05_cna_classifier/03_report/pdf_reports/all_sample_CNA_knowledge_reports.pdf
05_cna_classifier/03_report/clinician_reports/all_sample_clinician_driver_summaries.pdf
05_cna_classifier/06_knowledge/sample_knowledge_summary.tsv
05_cna_classifier/06_knowledge/sample_literature.tsv
05_cna_classifier/06_knowledge/knowledge_references.tsv
05_cna_classifier/07_pathology/pathology_concordance.tsv
05_cna_classifier/07_pathology/pathology_concordance_metrics.json

    06_workflow_summary¶

    Contains a compact path summary:

    06_workflow_summary/workflow_summary.txt

    This file is useful for downstream scripts because it records the actual paths for the run.
